# 常用标记语法

## 注释

```MarkDown
[comment]: this is a comment.
[//]: This is also a comment.
```

## 超链接

```MarkDown
[//]: 链接

[title](url)

[//]: 标题式链接

# [title](url)
```

## 文本换行

```MarkDown
this is a test code about line break

this is a test code about line break
```

## 引入图片

```MarkDown
![](https://i.loli.net/2018/03/26/5ab9081f17fe5.jpg)
```
